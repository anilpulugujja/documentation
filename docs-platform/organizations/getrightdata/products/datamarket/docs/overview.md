DataMarket enables secure commerce-grade data exchanges with entitlement, billing, and delivery automation built in.

## Value Props
- Multi-tenant catalogs with role-aware pricing.
- Marketplace storefront, search, and subscription flows.
- Delivery automations to Snowflake, S3, BigQuery, APIs, or webhooks.

Use the sidebar to configure the marketplace, publish offerings, and manage governance.
