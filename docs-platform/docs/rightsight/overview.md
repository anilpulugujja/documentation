RightSight brings premium streaming monitoring, anomaly detection, and AI-assisted investigations into a single workspace.

### Key Capabilities
- **Adaptive monitors** learn seasonal patterns using native ML.
- **AI Enrichment** produces root cause hypotheses from telemetry and metadata.
- **Unified dashboards** merge infra, application, and data KPIs.

### Architecture
1. Agents ship telemetry (metrics, traces, logs) to the ingestion API.
2. Streaming processors compute baselines and surface anomalies.
3. Insights sync back into DataTrust for governance context.
