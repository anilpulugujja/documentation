**Date:** February 10, 2026

### Highlights
- ✅ DataTrust Metadata Studio redesign with AI hints.
- 🚀 RightSight adaptive monitors GA.
- 🛍️ DataMarket billing with tiered usage.

### Breaking Changes
- Deprecated `v1beta` DataTrust domain endpoints; use `/v1/datatrust/domains`.

### Fixes
- Improved collector heartbeat reliability.
- Hardened webhook signature validation.

### Upgrade Checklist
1. Update collectors to `7.6.x` images.
2. Re-run Terraform module `v4.2.0` for new IAM policies.
3. Review alerts referencing legacy endpoints.
