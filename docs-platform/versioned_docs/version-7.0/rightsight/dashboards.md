Dashboards in RightSight are canvas-based and responsive by default.

## Creating a Dashboard
1. Click **Dashboards → New Dashboard**.
2. Choose a template (Golden Signals, Customer Journeys) or start blank.
3. Drag cards for metrics, text, image, or Markdown content.
4. Save with permissions (Org-wide, Domain, Private).

## Embedding
- Use the `iframe` embed code for internal portals.
- For customer-facing views, enable **Signed URLs** to restrict access.

## Theme Controls
Customize backgrounds, typography, card radius, and accent colors to match your brand.

:::tip Wide Layouts
Turn on **Wide Canvas** to unlock 1440px dashboards that match the docs layout aesthetic.
:::
